This is a personal website I created for Sheldon Landa who is a local NYC piano instructor. Sheldon's previous website was a few years old so he wanted to upgrade to a more modern looking site and something that was mobile friendly.

This project was my first freelance role. I learned a great deal about balancing client expectations with personal expectations. I also learned more about responsive web design and was also able to add some new photo editing skills with GIMP as this site included many images.
